## Rock-paer-scissors-game
This repositry consists of my Github project, It's title is "Rock, paper and scissors game".

## TITLE: "ROCK, PAPER AND SCISSORS GAME"

## OVERVIEW OF THE PROJECT:
 This project contains a simple rock, paper and scisors game where the user plays against the computer. This is a simple Python project where the user can play the Rock-Paper-Scissors game against the computer.
The program asks the user to enter their choice and then the computer randomly chooses one option.
After that, the program compares both choices and displays who won the round.

## FEATURES:
-User vs Computer gameplay
-Computer generates random choice
-Shows win / lose / draw results
-Runs in a loop until the user wants to exit
-Simple and easy to understand code

## TECHLOGIES AND TOOLS USED:
-Python 3
-random module (for computer choice)
-VS Code (code editor)
-Command Prompt /Terminal (to run the program)
-GitHub (for project submission)

## STEPS TO INSTALL AND RUN THE PROJECT:
1 Install Python 3 on your system
2 Download or clone this repository
3 Open the project folder
4 Open terminal inside the folder
5 Run the program using this command:
  python rps.py

## INSTRUCTIONS FOR TESTING THE PROJECT:
 1 Run the program in terminal
 2 When asked, type one of the following:
 rock
 paper
 scissors
 3 The computer will choose a random option
 4 The program will show:
 You Win
 You Lose
 Draw
 5 Repeat or exit when the program asks
 Make sure to test:
 All three inputs
 Invalid input (to see if the program handles it)
 Multiple rounds
