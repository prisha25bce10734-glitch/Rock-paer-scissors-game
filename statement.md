# Statement

## Problem Statement
The aim of this project is to create a simple Rock Paper Scissors game using Python.  
The user plays against the computer, and the computer randomly selects rock, paper, or scissors.  
The program compares both choices and shows the result (win, lose, or draw).

## Scope of the Project
- The game will run in the terminal/command prompt.
- The user will enter their choice through text input.
- The computer will make a random choice using Pythonâ€™s random module.
- The program will check all possible outcomes and display the correct result.
- The game can be repeated multiple times until the user wants to exit.

## Target Users
- Beginners who are learning Python.
- Students doing basic programming projects.
- Anyone who wants to play a simple text-based Rock Paper Scissors game.

## High-Level Features
- User input for rock, paper, or scissors.
- Computer generates a random choice.
- Result calculation based on game rules.
- Looping system to continue or stop the game.
- Simple and easy-to-understand structure.
